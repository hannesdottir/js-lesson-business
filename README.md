# Javascript 3 project - September 2020

## Create account - activate account - login - get customer list - add/remove from customer list - see customer detail

## API

ROOT URL: https://frebi.willandskill.eu/

LOGIN URL: ROOT URL + api-token-auth/

API_URL: ROOT URL + api/v1/

AUTH URL: ROOT URL + auth/

CUSTOMER LIST: API URL + customers

CUSTOMER DETAIL: API URL + customers/{id}/

LOGGED IN USER: ROOT URL + api/v1/me
